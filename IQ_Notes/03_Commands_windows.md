# VS Code Commands for Windows

This file contains common VS Code commands in a simple table format for Windows users.

| Action | Windows Command |
|---|---|
| Open Command Palette | Ctrl + Shift + P |
| Quick Open File | Ctrl + P |
| Save File | Ctrl + S |
| Save All | Ctrl + Shift + S |
| Open New Window | Ctrl + Shift + N |
| Close Current File | Ctrl + W |
| Close All Files | Ctrl + Shift + W |
| Search in Files | Ctrl + Shift + F |
| Find in Current File | Ctrl + F |
| Replace in Current File | Ctrl + H |
| Toggle Sidebar | Ctrl + B |
| Toggle Explorer | Ctrl + Shift + E |
| Toggle Integrated Terminal | Ctrl + ` |
| Split Editor | Ctrl + \ |
| Move to Next Editor | Ctrl + Tab |
| Move to Previous Editor | Ctrl + Shift + Tab |
| Go to Line | Ctrl + G |
| Comment/Uncomment Line | Ctrl + / |
| Format Document | Shift + Alt + F |
| Open Settings | Ctrl + , |
| Run Task | Ctrl + Shift + B |
| Open Extensions | Ctrl + Shift + X |

## Note

These shortcuts help you work faster in VS Code on Windows.
