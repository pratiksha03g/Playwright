# VS Code Commands for Mac

This file contains common VS Code commands in a simple table format for Mac users.

| Action | Mac Command |
|---|---|
| Open Command Palette | Cmd + Shift + P |
| Quick Open File | Cmd + P |
| Save File | Cmd + S |
| Save All | Cmd + Option + S |
| Open New Window | Cmd + Shift + N |
| Close Current File | Cmd + W |
| Close All Files | Cmd + Option + W |
| Search in Files | Cmd + Shift + F |
| Find in Current File | Cmd + F |
| Replace in Current File | Cmd + Option + F |
| Toggle Sidebar | Cmd + B |
| Toggle Explorer | Cmd + Shift + E |
| Toggle Integrated Terminal | Ctrl + ` |
| Split Editor | Cmd + \ |
| Move to Next Editor | Ctrl + Tab |
| Move to Previous Editor | Ctrl + Shift + Tab |
| Go to Line | Ctrl + G |
| Comment/Uncomment Line | Cmd + / |
| Format Document | Shift + Option + F |
| Open Settings | Cmd + , |
| Run Task | Cmd + Shift + B |
| Open Extensions | Cmd + Shift + X |

## Note

These shortcuts help you work faster in VS Code on Mac.
